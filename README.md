# 🎯 我的射击游戏

一个使用 Pygame 开发的 2D 射击游戏，是我学习 Python 和游戏开发的第一个项目。

## 🎮 如何运行
1.  确保已安装 Python 3.1及以上。
2.  安装 Pygame 库：在命令行输入 `pip install pygame`。
3.  在本文件夹中，运行命令 `python Square_shot1.2.py`。

## ✨ 当前功能
- 玩家移动（WASD）与射击（J键）
- 自动移动并反弹的敌人
- 分数系统
- **子弹数量限制与显示**（v1.2新增）
